# 🌦️ Weather App

A simple and responsive Weather App built using React and Vite that provides real-time weather information for any city worldwide.

---

## 🚀 Features

- 🔍 Search weather by city name  
- 🌡️ Real-time temperature updates  
- 💧 Humidity and 🌬️ wind speed display  
- 🌤️ Dynamic weather icons  
- 📱 Fully responsive design (mobile & desktop)

---

## 🛠️ Tech Stack

- React.js  
- Vite  
- JavaScript (ES6+)  
- OpenWeather API  
- CSS
